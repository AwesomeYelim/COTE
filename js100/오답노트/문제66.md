# 문제66. 블럭탑 쌓기

<mark style="background: pink">크기별로 준비된 블럭을 정해진 순서에 맞게 쌓아야한다</mark><br>

- 정해진 순서가 BAC 라면 A 다음 C 가 쌓아져야함
- B를 쌓지 않아도 A 와 C를 쌓을 수 있음
- B 다음 C가 될 수 있음


#### 체크사항

> 1. 블럭은 알파벳 대문자로 표기
> 2. 규칙에 없는 블럭이 사용될 수 있음
> 3. 중복된 블럭은 존재하지 x

```js
// 입력
탑 = ["ABCDEF", "BCAD", "ADEFQRX", "BEDFG", "EFGHZ"]
규칙 = "ABD"

// 출력
['가능', '불가능', '가능', '가능', '가능']
```
